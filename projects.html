Parrallel Password Breaker - October 2021 Python




Created a Python Program that uses parallel computing to break into passwords

The program began by taking in a random password. It then used threading and would generate random expressions. The expression were compared to the password entered until a match was found. If one thread was used, it would generate one password at a time alphabetically. If N threads were used, it would generate N threads alphabetically insted until the password was found, essentially dividing the total time by N if not for overhead.



Battleship Game - December 2021 C++




Created a user-playable battleship game using C++ with a probability based automated opponent

As far as game essentials go, the game was rather simple, a blank board was displayed on both sides. The game begins by asking the user to place their ships on the board, prompting them until they enter valid locations. Then the board updated to display a green s where their ships are located. After that they are asked to select locations to fire at. If it is a hit, the board updates to show a red H, otherwise it displays a blue M for miss. The game ends when the user quits or one person's ships are all sunk. Behind the scenes, the computer is constantly calculating the probability for every space on the board. For example, since the corners could only have any size ship placed in two ways, they have the lowest possibility on the board. If two locations have the same probability the computer randomly chooses between them. As the game goes on, the computer keeps recalculating these probabilities and firing at the high probability locations. When the computer lands a hit, it tests the locations around the hit to find the ship. When it lands two consecutive hits (which you can see in the photo) it will keep firing at that same ship until it is sunk.
